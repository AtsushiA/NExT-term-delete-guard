# NExT Term Delete Guard

カテゴリー、タグ、カスタムタクソノミーに紐づく記事がある場合、削除を防止するWordPressプラグインです。

## 機能

- 投稿が紐づいているタームの削除を防止
- 単一削除・一括削除の両方に対応
- Ajax削除リクエストのインターセプト
- 日本語エラーメッセージの表示

## 動作

1. 管理画面でターム（カテゴリー/タグ/タクソノミー）を削除しようとすると、そのタームに紐づく投稿があるかチェックします
2. 紐づく投稿がある場合、削除をブロックしエラーメッセージを表示します
3. 紐づく投稿がない場合、通常通り削除が実行されます

## インストール

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面の「プラグイン」から有効化

## 要件

- WordPress 5.0以上
- PHP 7.0以上
